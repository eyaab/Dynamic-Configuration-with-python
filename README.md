# Dynamic-Configuration-with-python
## Setup
For setting up the project run
* python3 -m venv venv
* pip install -r requirements.txt
## Development
### for windows: 
* set FLASK_APP=run.py
* flask run
### for linux:
* export FLASK_APP=run.py
* flask run
## run in dev mode to debug
set FLASK_ENV=development
flask run
